# Shopping Cart System
A full-featured E-Commerce Shopping Cart System developed as a BCA Final Year Project.
## Features
- User Registration & Login
- Admin Dashboard
- Product Management
- Category Management
- Shopping Cart
- Checkout System
- Order Management
- Customer Order History
- Bootstrap Responsive UI
## Technologies Used
- PHP
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository
2. Import the database into MySQL
3. Update `includes/db.php`
4. Run the project using XAMPP
